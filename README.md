# Prometheus Agent

AI-based agent to:
- Ingest metrics from Prometheus
- Train and detect anomalies
- Report and predict future anomalies

## Project Structure

```
prometheus-agent/
├── config/
├── data/
├── src/
│   ├── ingestion/
│   ├── storage/
│   ├── ml/
│   ├── reporting/
│   └── utils/
├── tests/
└── k8s/
```

Refer to [config/config.yaml](config/config.yaml) for configuration details.
